Consumer Lifestyle Trend Analysis (SQL)

Project Overview: This sample project analyzes high-velocity search trends in the lifestyle sector—specifically focusing on Vacation, Food, and Travel. By leveraging the Google Trends public dataset in BigQuery, I identified how these interests fluctuate, providing a foundation for demand-driven marketing and operational planning.

Technical Execution:
Platform: Google Cloud BigQuery

Techniques: Pattern matching (LIKE), time-series filtering (DATE_SUB), and data aggregation.
